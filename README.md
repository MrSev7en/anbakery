# Projeto Anhanguera 2024 - Professor Aléssio

A proposta do projeto é um sistema de padaria, em que o cliente pode fazer pedidos de produtos, e o sistema irá calcular o valor total do pedido.

## Como iniciar o projeto:

- Use o Live Server ou qualquer extensão que permita rodar um servidor local para abrir o arquivo **login.html**

```bash
code login.html
```

- Instale a extensão Live Server no Visual Studio Code (caso não tenha)
- Clique com o botão direito do mouse e selecione a opção "Open with Live Server"
- Acesse o projeto em:

```
http://127.0.0.1:5500/login.html
```

- Autentique-se com o usuário e senha:

```
Usuário: padeiro
Senha: 87654321
```
